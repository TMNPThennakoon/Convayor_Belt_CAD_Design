## ğŸ› ï¸ Technical Specifications

### Design Software
- **CAD Software**: SolidWorks
- **File Formats**: `.SLDPRT` (Parts), `.SLDASM` (Assembly)

### System Components

#### Power Transmission
- **DC Motor**: 77S series motor for reliable and controllable power delivery
- **Timing Belts**: XL series timing belts for precise synchronization
- **Pulleys**: Multiple pulley configurations for optimal belt tension and speed control

#### Structural Framework
- **T-Slotted Framing**: Modular aluminum extrusion system for flexible assembly and easy modifications
- **Custom Structural Parts**: Engineered components designed for specific load-bearing and mounting requirements

#### Motion Components
- **Ball Bearings**: High-quality bearings for smooth rotational motion
- **Corrosion-Resistant Components**: Specialized pulleys designed for harsh industrial environments

#### Fastening System
- **Grade 8 Steel Screws**: High-strength fasteners for critical connections
- **Zinc Plated Screws**: Corrosion-resistant fasteners for standard applications
- **Retaining Rings**: Internal retaining rings for secure component positioning

## ğŸš€ Getting Started

### Prerequisites

- **SolidWorks** (2019 or later recommended)
- Basic understanding of CAD modeling and assembly techniques

### Opening the Project

1. Clone this repository:
   ```bash
   git clone https://github.com/TMNPThennakoon/Convayor_Belt_CAD_Design.git
   cd Convayor_Belt_CAD_Design
   ```

2. Open the main assembly file:
   - Launch SolidWorks
   - Open `Assem1.SLDASM`
   - SolidWorks will automatically resolve all component references

3. Explore the design:
   - Use the FeatureManager design tree to navigate components
   - Rotate and zoom to examine the assembly from different angles
   - Check mates and constraints to understand component relationships
